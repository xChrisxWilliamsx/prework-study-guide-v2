HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is one of three main languages that makeup webpages.
  • HTML stands for HyperTextMarkupLanguage.
  • HTML is what appears on screen, it can be text, image, or the layout.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Separate multiple selectors by a comma to have the same code apple to each listed
  • The declaration block will have a property and an associated value to that property
  • As with many pieces of code, which include CSS, keep them separated to ensure everything is easy to view and manipulate

Git

  • git status: checks what branch you are on and if there are any changes staged for commit
  • git checkout -b [branch name]: creats a new branch and switches to it.
  • git restore [file name]: disregards changes to file.
  • git add -A: adds modifications in the current working branch to the staging area. The [-A] flag indicates that we want to add all changes.
  • git commit -m [file name]: gets the items in the staging area ready to commit to GitHub.
  • git push origin [branch]/[file name]: pushes changes, as a pull request, to GitHub for main codebase.
  • git pull origin [file name from repo i.e. main]: pulls/receives a branch modifications onto local environment[your computer].
